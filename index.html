Your AI coding agent can run any command it wants, right now.

One click through a permission dialog you stopped reading, and it just did something you did not mean to allow. AethelHook routes every risky action from Claude Code, Codex, or Antigravity to your phone first. You decide, not a popup you clicked past.

Windows PC and Android phone required. No account, no subscription.

AI coding agents can execute shell commands, overwrite files, and install packages on their own. Most IDEs show a quick allow or deny popup on the same screen you are already working in, which is easy to click through on autopilot. AethelHook forces that decision onto a second device, so approving something dangerous takes a deliberate action, not a reflex.

What it does

One gateway, three IDEs, and a phone in your pocket.

Works with Claude Code, Codex, and Antigravity

Each IDE is wired in through its own native hook mechanism, all routed through the same approval gateway.

Real time, end to end encrypted

Approvals travel over a TLS connection with certificate pinning. No cloud relay and no third party push service sits in the middle.

Five ways to answer

Allow once, always allow for this project, always allow everywhere, deny, or deny with a reason fed back to the agent.

Plans and questions too

Claude Code's plan review and clarifying questions are answered from your phone, not just tool calls.

Session Access

Send a prompt from your phone and kick off a headless run on your PC. Each project keeps its own resumable conversation.

Secure by default

Devices pair over a one time QR code. Every device gets its own token, and nothing is stored in plain text.

Download

Both installs are free, and the full source is public on GitHub.

Windows

Installs the background service and the tray app.

Download for Windows
You will likely see a SmartScreen warning. This installer is not code signed yet, since the usual free signing programs require a longer release history than this project has so far. Click More info, then Run anyway. The full source is public, so you can check exactly what it does first.

Android

Sideloaded APK, not on the Play Store yet.

Download for Android
Android will warn about an unknown source. That is expected for an app that is not on the Play Store yet. Allow the install to continue, then open the app and scan the pairing QR code shown on your PC.
